# node-test

1) Update variables in .env:
  PORT - port where application will run
  DATABASE_SRV_URI - address of database at https://cloud.mongodb.com

2) Run in console:

```
cd ./src
npm i
npm start
```

### Options
Run for development:
```
npm run dev
```
Run for build and run:
```
npm start
```
Run for remove build files:
```
npm clear
```
